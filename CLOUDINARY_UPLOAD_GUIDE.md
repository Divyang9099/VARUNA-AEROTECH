# Upload Mockup Images to Cloudinary

## Step 1: Get Your Cloudinary Credentials
1. Go to https://cloudinary.com/console
2. Copy your:
   - Cloud Name: `ddsjqtxik` (already have this)
   - API Key
   - API Secret

## Step 2: Install Cloudinary Package
```bash
npm install cloudinary
```

## Step 3: Update upload-mockups.js
Replace `YOUR_API_KEY` and `YOUR_API_SECRET` in the script with your actual credentials.

## Step 4: Run the Upload Script
```bash
node upload-mockups.js
```

## Step 5: After Upload
The script will create `cloudinary-mockup-urls.json` with all the URLs.
